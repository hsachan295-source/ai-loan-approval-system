# 🏦 Loan Default Prediction System

## 📌 Overview

This project predicts whether a loan will be approved or defaulted using Machine Learning (Gaussian Naive Bayes). It helps financial institutions assess credit risk efficiently.

## ☁️ Deployment (Streamlit Cloud)

### Step-by-Step:

1. Push code to GitHub
2. Go to: https://streamlit.io/cloud
3. Click **New App**
4. Select your GitHub repository
5. Choose:

   * Branch: `main`
   * File: `app.py`
6. Click **Deploy**

✅ Your app will be live in 2–3 minutes
